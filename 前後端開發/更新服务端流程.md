## 更新

### 前端

- 在cmd里
1. 进入前端项目
2. 安装依赖
3. 构建项目
4. 上传、覆盖旧文件
```
cd E:\ShuoWen\V0.2\yuzongweb
npm install
npm run build
cd D:\install_route\putty\
pscp.exe -pw Ab123456 -r E:\ShuoWen\V0.2\yuzongweb\dist\* root@47.108.117.70:/usr/share/nginx/html/
```


### 后端

- IDEA里
1. 进入Maven
2. 生命周期
3. 先点clean，再点package

- 在cmd里
```
cd D:\install_route\putty\
pscp.exe -pw Ab123456 "E:\ShuoWen\V0.2\YuZong\target\yuzong-0.0.1-SNAPSHOT.jar" root@47.108.117.70:/home/app/
```

- 在服务器里 
#### 如果没有变化名称
1. 进入应用目录
2. 重启服务
3. 检查状态
```
cd /home/app
sudo systemctl restart springboot-app
sudo systemctl status springboot-app
```

#### 如果变化了包名
1. 检查现在运行的包
2. 编辑服务配置文件 （vim使用i键进入编辑模式，编辑完成Esc键退出编辑模式，输入:wq）
3. 重新加载服务
4. 重启服务
5. 检查状态
```
sudo systemctl status springboot-app
sudo vi /etc/systemd/system/springboot-app.service
sudo systemctl daemon-reload
sudo systemctl restart springboot-app
sudo systemctl status springboot-app
```
